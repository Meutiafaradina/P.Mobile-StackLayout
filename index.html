Kode Kivy(StackLayout)


Kode Python:

from kivy.uix.button import Button from kivy.uix.stacklayout import StackLayout class myApp(App): def build(self): s_layout= StackLayout(orientation='rl-bt') btn1=Button(text='btn1', size_hint=(0.2, 0.2)) btn2=Button(text='btn2', size_hint=(0.2, 0.2)) btn3=Button(text='btn3', size_hint=(0.2, 0.2)) btn4=Button(text='btn4', size_hint=(0.2, 0.2)) s_layout.add_widget(btn1) s_layout.add_widget(btn2) s_layout.add_widget(btn3) s_layout.add_widget(btn4) return s_layout myApp().run()

Penjelasannya:

1 Import Button

Mengimpor Button dari kivy.uix.button. Kelas ini menyediakan widget tombol yang bisa ditekan untuk interaksi pengguna.

2 Import StackLayout

Mengimpor StackLayout, layout manager yang secara otomatis menumpuk dan melayout widget sesuai orientasi (horizontal/vertical, dll.).

3 Definisi kelas myApp

Membuat kelas myApp yang merupakan turunan dari App, yang menjadi entry point utama aplikasi Kivy.

4 Metode build()

Fungsi utama yang wajib diimplementasi. Mengembalikan root widget yang akan dirender ke layar saat aplikasi dijalankan.

5 Buat StackLayout

Instansiasi StackLayout dengan orientasi 'rl-bt' (right-to-left, bottom-to-top) untuk mengatur aliran widget.

6 Buat Button 1

Membuat tombol dengan teks 'btn1', ukuran relatif 20% dari kontainer untuk lebar dan tinggi.

7 Buat Button 2

Button kedua dengan teks 'btn2', ukuran sama untuk konsistensi desain.

8 Buat Button 3

Button ketiga dengan teks 'btn3'. Layout akan mengatur penempatannya sesuai orientasi StackLayout.

9 Buat Button 4

Button keempat dengan teks 'btn4'. Semua tombol dibuat seragam untuk tata letak yang rapi.

10-13 Tambah ke Layout

Keempat tombol ditambahkan ke s_layout satu per satu dengan add_widget(), yang menentukan urutan dan posisi di layout.

14 Kembalikan Layout

Mengembalikan s_layout ke Kivy sebagai root widget aplikasi. Layout inilah yang akan dirender di window aplikasi.

15 Jalankan Aplikasi

Memanggil myApp().run() untuk memulai event loop Kivy dan menampilkan GUI ke pengguna.
